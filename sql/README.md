# SQL Practice

SQL exercises and query optimization focused on data engineering 
patterns and interview preparation.

## Purpose

Build SQL fluency for data engineering roles through:
- **Fundamentals**: JOINs, aggregations, window functions
- **Optimization**: Query performance, EXPLAIN plans, indexing
- **Interview prep**: Common DE interview patterns

## Structure

- `art-of-postgres/`: Exercises from "The Art of PostgreSQL"
- `data-lemur/`: SQL interview questions
- `query-optimization/`: EXPLAIN practice and performance tuning
