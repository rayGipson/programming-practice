# Bash Practice - Unix/Linux Curriculum

> **Purpose**: Systematic progression from bash fundamentals to advanced system programming. 
> Building fluency in bash, text processing, system administration, and automation before 
> transitioning to Python data engineering projects.

## Learning Objectives

1. **Beginner**: Navigate filesystem, manage files, write basic scripts
2. **Intermediate**: Text processing, process management, automation with cron
3. **Advanced**: System internals, service management, advanced bash patterns
4. **Supplemental**: Regex mastery, Git CLI proficiency, dotfile customization
